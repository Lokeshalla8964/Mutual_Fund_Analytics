# Mutual Fund Analytics

## Project Overview
This project analyzes Indian mutual fund data using Python and Power BI. It includes data ingestion, cleaning, exploratory data analysis, performance metrics, advanced risk analytics, and dashboard visualization.

## Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Daily Returns
- CAGR (1Y, 3Y, 5Y)
- Sharpe Ratio
- Sortino Ratio
- Alpha & Beta
- Maximum Drawdown
- Value at Risk (VaR)
- Conditional Value at Risk (CVaR)
- Rolling Sharpe Ratio
- Investor Cohort Analysis
- SIP Continuity Analysis
- Sector HHI Analysis
- Mutual Fund Recommendation System

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- SQLite
- Power BI
- Jupyter Notebook

## How to Run

1. Install dependencies

```
pip install -r requirements.txt
```

2. Open the notebooks.

3. Run all cells.
